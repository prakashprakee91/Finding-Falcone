How to execute code
-------------------
1) run 'npm install' to install all dependencies.

2) run 'npm run start' to execute code.

How to run test
---------------
1) In test.ts file replace app.component with name of the component you want to 
test e.g. suppose you want to test successmessge component replace "app.component"
with "successmessage.component".

2) After replacing run "ng test"/ "npm run test".

How to See Coverage
--------------------
1) Do the same step1 as in case of test.
2) then run "ng-test --code--coverage", this will create a coverage file in which 
you can find the code covered in test cases for a component.
